# Lick Links - Link Checker

### About

This is a CLI link checker written in Typescript.

### Installation

```bash
npm i -g lick-links
```

### Usage

#### npx

No install needed

```bash
npx lick-links <siteUrl> [options]
```

#### If installed

```bash
lick <siteUrl> [options]
```

### Help

```bash
lick --help
```
